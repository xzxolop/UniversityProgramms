# UniversityProgramms

Репозиторий с набором независимых учебных проектов. Ниже перечислены все поддиректории и кратко описано, на чем они написаны и что делают.

## Проекты

- `asd/List/Lab-02` - C++ / Visual Studio. Реализация пользовательского двусвязного списка `MyList` и отдельный проект с тестами для проверки его методов.
- `asd/Tree/Lab_3.1` - C++ / Visual Studio. Бинарное дерево поиска с обходами, сравнением деревьев и поиском минимального и максимального элементов.
- `caos/lab14` - C++. Программа-планировщик задач с очередью приоритетов и отдельным потоком обработки задач.
- `CppLibrary/VoroninLib` - C++ / Visual Studio. Библиотека с набором учебных функций и структур данных: массивы, строки, файлы, списки, деревья и рекурсивные алгоритмы.
- `CsLibrary` - C#. Аналогичная учебная библиотека с набором вспомогательных функций и алгоритмов.
- `maple` - Maple. Набор работ по вычислительной математике: интегрирование, интерполяция, решение линейных и нелинейных уравнений, метод наименьших квадратов и связанные тестовые варианты.
- `Ruby/koans` - Ruby. Набор Ruby Koans для изучения языка через последовательные задания и проверки.
- `web/lab1 lab2` - HTML. Учебная web-страница с табличной версткой и набором изображений/ссылок.
- `web/lab3` - HTML. Страница с таблицей, блоками диалогов, медиа-вставками и примерами ссылок между страницами.
- `web/lab4` - HTML. Учебная страница с диалогами, таблицей, списками, выпадающим списком и другими элементами разметки.
- `web/lab6/flex` - HTML/CSS. Макет страницы, сверстанный на flexbox.
- `web/lab6/grid` - HTML/CSS. Макет страницы, сверстанный на CSS Grid.
- `web/lab7/react-files` - JavaScript / React. Простейшее приложение на Create React App с базовым стартовым шаблоном.
- `web/lab7 bros` - JavaScript / React. React-приложение с собственной моделью и компонентом `App`, который рендерит этот компонент.
- `Clock` - Проект "Часы" - это простое WPF приложение, реализующее отображение текущего времени в окне приложения.
Полностью написаны на PascalABC.NET

Уникальность этого проекта заключается в том, что перед разработчиком была поставлена цель написать код как можно хуже.
В итоге получилось 2378 строк.

Сердцем программы являются 180 функций, которые описывают все возможные положения часовой, минутной и секундной стрелки (по 60 на каждую).
После этого в основной функции с помощью оператора выбора под текущее время на компьютере вызывается конкретная функция.
Для реализации этого потребовалось 1059 строк.
Ежесекундно наша программа под текущее время на компьютере вызывает необходимую функцию отрисовки. Это может приводить к "мерцанию часов".
